# Beanie Gs 

## Overview
If Michael Jackson wears beanies he would've worn these.
This app lets users browse, search and buy their favorites beanies and other products in the future.

## Technologies used
  Node.js
  Reactstrap
  auth0
  Stripe
  API
  Express
  React
  Mongoose
  Redux
  Fontawesome
  morgan
  helmet for security
## Installation
  npm i
  npm start
